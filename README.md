# Css flexbox nav bar
A Navigation Bar using the CSS Flexbox Method. A flexbox an element that can house multiple elements aligning them to one-dimension — either a row or a column, allowing for more dynamic resizing. The nav bar element gives an overview of the possible pages you can access on the website.

## About
It contains and displays logo, work, play, about, contact pages

## What I learned
You can create custom properties / variables in :root that can be referenced throughout the CSS. You only need to change it once for it to update throughout the css. 

Media query's can be just one block with multiple classes and elements in them. You can have multiple querys for multiple breakpoints. 

Instead of class selectors I can use child elements such as "li:first-child".

Margin-auto: right can create a gap in the flexbox pinning elements to specific positions.
